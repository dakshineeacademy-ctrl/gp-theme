A premium, high-performance WordPress theme for commodity rates (Gold, Silver, Oil) with live charts and dark mode.

== Description ==

22K Gold Rate is a lightweight, high-performance WordPress theme designed specifically for financial and commodity rate websites. It features built-in support for live price charts (Gold, Silver, Oil) using Chart.js, a modern "Premium" design aesthetic, and a native Dark Mode toggle.

Key Features:
*   **Live Price Charts**: Automatically displays interactive charts for Gold, Silver, and Oil pages.
*   **Dark Mode**: Native toggle with user preference persistence.
*   **Performance**: Zero dependencies, no heavy frameworks, optimized for Core Web Vitals.
*   **SEO Optimized**: Semantic HTML5 structure and Schema markup ready.
*   **Page Builders**: Fully compatible with Elementor and Block Editor (Gutenberg).

== Installation ==

1.  Upload the `Theme` folder to the `/wp-content/themes/` directory.
2.  Activate the theme through the 'Themes' menu in WordPress.
3.  Create pages with slugs `gold`, `silver`, or `oil` to automatically display the respective charts.
4.  Set a static homepage in Settings > Reading to use the custom front-page layout.

== Frequently Asked Questions ==

= How do I change the chart data? =
The current version uses mock data for demonstration. You can connect it to a real API in `assets/js/main.js`.

= Does this support Elementor? =
Yes, this theme is fully compatible with Elementor and other page builders.
